# The Game

Welcome. I salute you to the intriguing world of The Silver Case hacking/translating.

In this guide/manual I'll take you through lots of the discoveries and intricacies this game has to offer in order to be translated and meddled with. Let's start with some details about the game:

The Silver Case was developed in Unity version 5.4.1p4. Fairly old by now and long out of support, which could mean that the game could be harder to translate as time goes by. Hence, this guide aims to show you comfortable ways to translate it despite the passage of time.

More info on: https://www.pcgamingwiki.com/wiki/The_Silver_Case
