# Introduction

Welcome to The Silver Case modding guide. This section provides an overview of how to modify and translate The Silver Case, one of the most intricate text adventure games.

The Silver Case uses Unity version 5.4.1p4 and stores its content in various formats:

- **JSON-formatted text files** for dialogue and chapter text
- **Binary files** within Unity asset containers for images, videos, and other assets
- **Game logic** in compiled C# code (Assembly-CSharp.dll)

To get started with modding, you'll need to use Unity asset extraction tools such as UABEA, UnityEX, or AssetStudio. These tools allow you to extract and reimport game assets while maintaining the proper file formats and structure.

This guide covers:

- **Dialogue and Chapter text** - Modifying game dialogue using JSON editors
- **Images** - Editing texture and image assets
- **Videos** - Working with video files
- **Contact Objects & Characters** - Modifying interactive elements and character data
- **Interact Objects** - Editing interactive object properties
- **Fonts** - Customizing game fonts
- **Menu Texts** - Translating menu interface text

Each section provides detailed instructions for extracting, editing, and reimporting the specific asset types.
